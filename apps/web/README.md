# @zeno/web

Next.js 16 dashboard with three portals: regulator (SPCB/CPCB), industry (facility operators), and public.

Powered entirely by Hedera Mirror Node REST API — no external database.

## Run

```bash
npm run dev -w apps/web
# http://localhost:3000
```

## Stack

- Next.js 16.1.6
- shadcn/ui
- React-Leaflet 5.0 (facility maps)
- Recharts 3.8 (compliance charts)
- next-intl 4.8.3
